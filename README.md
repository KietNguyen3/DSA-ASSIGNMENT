# Data Structures & Algorithms Projects (C++)

## Overview
This repository contains programming assignments for the Data Structures and Algorithms course.
The projects focus on implementing fundamental data structures and applying algorithmic techniques
using modern C++.

## Implemented Data Structures
- Dynamic arrays
- Doubly linked lists
- Hash tables
- Binary heaps / priority queues
- Trees (including N-ary Huffman Trees)

## Project Highlights
- Built an inventory management system using custom data structures
- Extended the system with N-ary Huffman Tree–based compression for efficient data storage
- Implemented generic data structures using templates and iterators
- Managed memory manually without relying on STL containers
- Passed both public and hidden test cases

## Technologies
- C++ (C++17)
- Object-Oriented Programming
- Templates and Iterators
- Memory Management

## Notes
These projects were completed as part of academic coursework and emphasize correctness,
efficiency, and clean code structure rather than UI or production-level features.
